
# Sense AI

Sense AI is an advanced AI-powered technology to sense any phishing site, dark pattern and malicious web-behaviour and protect it's users.

## Project Info

**Repository URL**: <YOUR_GIT_URL>

## Getting Started

You can work on Sense AI using your preferred IDE or directly on GitHub. To get started locally:

```sh
# Step 1: Clone the repository using your Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd sense-ai

# Step 3: Install the necessary dependencies (using npm, pnpm, or yarn).
pnpm install

# Step 4: Start the development server with auto-reloading and instant preview.
pnpm run dev
```

You can also edit files directly in GitHub or use GitHub Codespaces for a cloud-based development environment.

## Technologies Used

## Deployment

You can deploy Sense AI using your preferred platform (Vercel, Netlify, etc.).

1. Build the project:
	```sh
	pnpm run build
	```
2. Deploy the contents of the `dist` folder to your hosting provider.

## Custom Domain

If your deployment platform supports custom domains, follow their documentation to connect your domain to your deployed Sense AI instance.
